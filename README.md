## 📖 Overview
This project serves as the user-facing dashboard for the Content Management platform. Built with a focus on clean UI/UX and lightweight performance, it operates completely independently of the backend lifecycle. It uses modern JavaScript `async/await` patterns to request live JSON payloads from the cloud API and renders them dynamically into the DOM.

## ✨ Features
* **Live Data Hydration:** Fetches real-time records directly from MongoDB Atlas via the API.
* **Asynchronous Fetching:** Zero page reloads; utilizes native JavaScript Fetch API.
* **Decoupled Architecture:** Communicates cross-origin with an isolated Express server.
* **Responsive Layout:** Minimalist interface optimized for desktop, tablet, and mobile.

## 🛠️ Tech Stack
* **Frontend:** HTML5, Modern JavaScript (ES6+)
* **Styling:** CSS3 / Flexbox & Grid
* **API Communication:** Native `fetch()` API
* **Cloud Deployment:** Vercel
